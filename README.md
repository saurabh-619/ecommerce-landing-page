# Ecommerce site's landing page

This is a Next.js + Framer motion based ecommerce site landing page with cool animations. 

## Libraries

- Next.js = Frontend
- Framer motion = animations
- react_icons = Feather/Font awesome icons. 
 
## Demo
  [Web - Ecommerce site](https://ecommerce-landing-page-snowy.vercel.app/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/saurabh-619/ecommerce-landing-page
```

Go to the project directory

```bash
  cd ecommerce-landing-page
```

Install dependencies

```bash
  yarn
```

Start the app

```bash
  yarn dev
```
